Voice Input Guide

  • Keyboard shortcuts: Press Alt+M (Windows/Linux) or Cmd+M (Mac) to toggle recording. Alternatively, use Ctrl+Space (Windows/Linux) or Cmd+Space (Mac).
  • Real-time transcription: Text appears in the input field as you speak.
  • Stop recording: Press the same shortcut again to stop recording.
  • Submit: After recording, press Enter to submit or edit the text first.
  • Browser support: Works in Chrome, Edge, and Safari. Requires microphone permission.
